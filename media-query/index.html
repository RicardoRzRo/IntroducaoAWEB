The Road to Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broken CSS support, and abandoned browsers.

We must clear the mind of the past. Web enlightenment has been achieved thanks to the efforts of the W3C, WaSP, and major browser creators.

The CSS Zen Garden invites you to relax and meditate on important lessons. Become one with the web.

So What is This About?

The Zen Garden aims to show the power of CSS. Click a design to see how the same HTML gets completely restyled. Yes, really.

CSS allows full control over styling. Designers and coders alike have contributed to the beauty of the web.

Participation

You are modifying only the CSS, not the HTML. Download the sample files and upload your design when ready.

Benefits

This project serves as both inspiration and a learning tool for those working on or entering web development.

Requirements

Please focus on widely supported CSS. Submit original work, and license it under Creative Commons so others can learn.
